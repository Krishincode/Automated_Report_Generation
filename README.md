## Automated Sales Report Generator

An automated reporting system built using Python that processes raw sales data, generates summary insights, visualizes performance, and exports a formatted Excel report.

## Project Overview

This project reads sales data from a CSV file, performs data cleaning and aggregation, creates visualizations, and generates a structured Excel report automatically.

The goal is to reduce manual reporting effort and improve efficiency in handling repetitive business data tasks.



## Features

- Reads raw sales data from CSV

- Handles duplicate and missing values

- Generates category-wise sales summary

- Creates sales visualization (bar chart)

- Exports formatted Excel report

- Saves report with timestamp

## Technologies Used

*Python
*pandas
*matplotlib
*openpyxl
